# Sales Transaction Data Wrangling

## Overview

This repository contains a complete data cleaning and wrangling workflow performed on a sales transaction dataset containing 10K+ records.

The objective was to transform raw and inconsistent transactional data into a clean, structured, and analysis-ready dataset.

## Tasks Performed

* Missing value handling
* Duplicate removal
* Data standardization
* Outlier detection using IQR
* Final dataset validation

## Technologies Used

* Python
* Pandas
* NumPy
* Jupyter Notebook

## Dataset Features

* Customer Information
* Product Details
* Transaction Data
* Payment Information
* Regional Sales Information

## Output

A fully cleaned and validated dataset ready for analysis and visualization.
